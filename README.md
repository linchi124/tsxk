# 吞噬星空 · 转盘游戏

一款基于《吞噬星空》世界观的网页转盘游戏。随机穿越到原著的 6 个时空节点，通过 60 次转盘决定你的命运走向。

## ✨ 特色

- **6 个时间阶段**：地球时期 → 罗峰宇宙闯荡 → 域外战场 → 原始星 → 宇宙海 → 界兽危机
- **60 次转盘**：每阶段 10 次，决定你在每个时代的遭遇
- **加权概率**：阶段 1（35%）、阶段 2（30%）、阶段 3（15%）、阶段 4（10%）、阶段 5（8%）、阶段 6（2%）
- **因果系统**：阶段之间、转盘之间的结果相互影响
- **境界成长**：从行星级到永恒真神的完整修炼体系
- **宝物系统**：弑吴羽翼、星辰塔、黑神套装、血洛晶等原著宝物
- **特殊经历**：触发额外转盘，获得奇遇、传承或秘境
- **9 种结局**：从与罗峰联手到被界兽吞噬，多种命运分支
- **响应式**：支持桌面端、平板和手机端

## 🎯 游戏玩法

1. **第 1 次转盘**：选择你穿越到的时代节点
2. **阶段内 10 次转盘**：在当前阶段中遭遇各种事件
3. **因果流转**：阶段最后一次转盘的结果影响下一阶段的内容
4. **结局判定**：60 次转盘后，根据积累的境界、宝物、机遇判定 9 种结局之一

## 🔧 技术栈

- **纯前端**：HTML5 + CSS3 + JavaScript
- **转盘绘制**：Canvas 2D
- **响应式**：CSS Grid + Flexbox + Media Queries
- **无框架**：原生 JS，无需构建工具
